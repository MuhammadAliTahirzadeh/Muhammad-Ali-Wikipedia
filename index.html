Haqqımda

Tahirzadə Məhəmməd Əli — Azərbaycanlı tələbə və gələcək proqramçı. 2007-ci il 30 iyun Azərbaycan dövlətində Lerik şəhərində anadan olmuşdur. Hazırda Azərbaycan Dövlət İqtisad Universitetində (UNEC) ali təhsil alır. Əsas maraq sahəsi Front-End veb proqramlaşdırmadır. O, özünü bu sahədə inkişaf etdirməyə və gələcəkdə peşəkar veb developer olmağa çalışır.

Təhsil

Məhəmməd Əli 2013 cü ildən etibarən Bakı şəhərində yerləşən 193 nömrəli tam orta məktəbdə təhsil almış və burada 1-ci sinifdən 10-cu sinifə qədər oxumuşdur. Təhsilinin son iki ilini isə 212 nömrəli tam orta məktəbdə davam etdirmişdir. Məktəbi 613 bal nəticəsi ilə başa vurmuş və Azərbaycan Dövlət İqtisad Universitetinin (UNEC) İnformasiya Texnologiyaları ixtisasına qəbul olunmuşdur.

O, eyni zamanda Coders Academy tərəfindən təşkil edilən “Müasir Proqramlaşdırma və Süni İntellekt” təqaüd proqramına seçilərək bu sahədə əlavə biliklər əldə etməyə davam etdirir.

Əsasən self-study (öz-özünə öyrənmə) üsuluna üstünlük verir və fərdi şəkildə proqramlaşdırma bacarıqlarını inkişaf etdirməkdə davam edir.

İT Bacarıqları

Məhəmməd Əli İT sahəsinə marağını 9-cu sinifdə kəşf etmiş və ilk olaraq HTML və CSS öyrənməyə başlamışdır. İmtahanlara hazırlaşdığı dövrdə bu sahədən bir müddət uzaq qalmış, lakin universitetə qəbul olduqdan sonra proqramlaşdırma üzrə biliklərini artırmağa davam etmişdir.

Hazırda əsasən Front-End sahəsində özünü inkişaf etdirir. HTML5,CSS3,Python və Javascript üzrə fundamental biliklərə malikdir. Platforma və alətlər baxımından Git versiya nəzarət sistemi, GitHub kod hostinq platforması və Visual Studio Code (VS Code) redaktoru ilə işləmə bacarıqlarını inkişaf etdirir.

Hobbilər və Maraqlar

Məhəmməd Əli yalnız İT sahəsində deyil, müxtəlif intellektual və mənəvi sahələrdə də fəal maraq göstərir. O, paralel olaraq Al-Mustafa Open University-də təhsil alaraq İslam dini və dini elmlər sahəsində biliklərini inkişaf etdirir. İslam dini,Teologiya,Əxlaq mövzusunda kitablar oxumaq və araşdırmalar aparmaq onun əsas intellektual maraqlarındandır.

Eyni zamanda fəlsəfə və psixologiya sahələrinə maraq göstərir, bu sahələrdə kitab oxuyur və şəxsi araşdırmalar aparır. Mədəni zövq baxımından daha çox dram və tarixi mövzularda filmlərə üstünlük verir.

Günlük həyatında fiziki sağlamlığa da diqqət yetirir və müntəzəm olaraq idmanla məşğul olur.
